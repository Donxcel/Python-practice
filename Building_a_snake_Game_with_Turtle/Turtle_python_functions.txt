turtle.Screen()  -> used to create a window
window.title() -> set's the name of the program
window.setup(width = X, height = Y ) -> define the x and y coordinates for scrren size
window.bgcolor("") -> used to set the background color of the window
window.listen() -> listen to key presses
window.onkeypress(function,"key") -> bound the function to a corresponding key
window.update -> keeps updating the screen continuously
window.tracer(a) -> turn animation on or off
